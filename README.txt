
# Employee Reward Predictor (Streamlit)

## Setup
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Run the app
```bash
streamlit run reward_predictor_app.py
```

Place `employee_reward_data.csv` in the same folder or upload it via the sidebar.
